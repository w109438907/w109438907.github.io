2018年终总结

2018年终总结

一.回顾2018

关于工作：

2018供职于一家以直播为主要业务的公司，大体内容如下：

  1. 对音视频方面的知识有了一定的了解
    主要涉及的是编解码，Android Api 、ffmpeg以及OpenGL相关知识。
    已经有大神对此做出过汇总，在此记录一下：
    https://www.cnblogs.com/renhui/p/7452572.html
  2. Android 人脸识别
    主要依赖Google的Vision库和openCV
  3. Android 连接 UVC Camera
  4. Android Vpn Service 相关内容

关于学习：

  1. 业余时间阅读了一些技术类书籍如《 Android开发艺术探索 》、《 Android高级进阶 》、《 C++ Primer 》、《 CSAPP 》等。其中，《 C++Primer 》为快速阅读，主要为NDK开发巩固一下知识。但是《 CSAPP 》并没有完全读完，主要还是卡在了汇编上。虽然大学专业是软工，但是没有汇编课程，编译原理安排了八周课程，只教了后半本书，当时课上得异常痛苦，现在对汇编也是一知半解。不过有问题未尝不是好事，查漏补缺，《 CSAPP 》还是需要继续啃下去。
  2. 如今的大数据和人工智能领域非常火爆。今年在 Coursera 上学习了吴恩达的 machine learning 课程，算是对机器学习有了一点了解。即使目前不从事AI相关工作，但是还是要学习、了解的。
  3. 今年也去现场感受了一下谷歌开发者大会。主要还是在推销自己的技术和框架，比如 Flutter，Android with Kotlin，TensorFlow 等等。总体来说，第一次经历这种发布会，感觉还不错。
  4. 在极客时间上购买了几个课程，上下班路上正好听一听，还是有收获的。

二.2019小目标

  1. 坚持记录自己的所学，所想

    2018年下半年就立下的flag，但是真到了开始的时候，却又不知道写什么，如何写（排版啊，文字描述啊，都不是我擅长的，慢慢加油吧！）。万事开头难，今天勉强算是个开始吧。

  2. 学习 Tensorflow

    谷歌出品机器学习框架，在开发者大会上听着还不错。目前在 Github 上已经斩获近12万的star，维护和开发速度很快，即将发布2.0版本，值得一学。

  3. 学习 Flutter

    虽然目前从事 Android 开发工作，但是我感觉整个前端的 “融合” 会是一个趋势，比如之前 Facebook 推出了 React Native，而今 Google 又在大力宣传自己的 Flutter。科技巨佬们好像都在朝这个方向做这个事情。对于个人来说，如果能在前端的某一方面（比如 Android、Ios ）做到顶尖，那当然是很好的，但是如果能够让自己越来越全面，想必也是一件美事。

  4. 基础知识

    在过去的工作和学习之中，还是能够发现自己在基础上的不足的。目前自己感到的薄弱点还是算法以及英文能力。在算法方面，计划把以前看的一些经典书籍再巩固一下，相信以自己现在的实力，会有跟初读时不一样的收获。至于英文，目前还是局限于，有问题时坚持 Google 和 坚持阅读英文文档，后续有什么想法的话，我再回来补全。
